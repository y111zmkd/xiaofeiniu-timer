# 小飞牛计时

面向飞行学员的手机时间计算器，包含：

正式网址：[https://y111zmkd.github.io/xiaofeiniu-timer/](https://y111zmkd.github.io/xiaofeiniu-timer/)

- 航段时长计算：使用 24 小时制滚轮计算开始时间到结束时间的时长。
- 航时汇总运算：将两段航时相加或相减。
- 手机桌面安装：支持 iPhone 和安卓添加到主屏幕。
- 离线使用：首次联网完整打开后，可以断网重新打开并计算。

## 手机安装

- iPhone：先删除旧的桌面入口，用 Safari 打开正式网址，看到“已可离线使用”后点击“分享”，选择“添加到主屏幕”。
- 安卓：先删除旧的桌面入口，用 Chrome 打开正式网址，看到“已可离线使用”后在浏览器菜单中选择“安装应用”或“添加到主屏幕”。

第一次安装和加载必须联网。如果 Wi‑Fi 无法访问 GitHub Pages，请使用手机流量打开并等待“已可离线使用”。添加到主屏幕后，保持联网从新桌面图标完整打开一次；完全关闭后即可在飞行模式或不稳定 Wi‑Fi 下重新打开。以后更新时联网打开一次、完全关闭、再打开即可使用新版。

也可以直接扫描仓库根目录的 `小飞牛计时-分享二维码.png` 打开正式网址。

## 本地运行

```powershell
python -m http.server 8000 --bind 0.0.0.0
```

同一局域网内的手机可通过电脑局域网 IP 和端口 `8000` 访问。

## 测试

```powershell
node --test
```

## 技术结构

应用为纯静态 HTML/CSS/JavaScript，不使用服务器、数据库、账号或云同步。`manifest.webmanifest` 提供手机安装信息，`sw.js` 负责离线缓存。
