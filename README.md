# 小飞牛计时

面向飞行学员的手机时间计算器，包含：

- 航段时长计算：使用 24 小时制滚轮计算开始时间到结束时间的时长。
- 航时汇总运算：将两段航时相加或相减。
- 手机桌面安装：支持 iPhone 和安卓添加到主屏幕。
- 离线使用：首次联网完整打开后，可以断网重新打开并计算。

## 手机安装

- iPhone：用 Safari 打开正式网址，点击“分享”，选择“添加到主屏幕”。
- 安卓：用 Chrome 打开正式网址，在浏览器菜单中选择“安装应用”或“添加到主屏幕”。

第一次安装和加载必须联网。之后可以从桌面的“小飞牛计时”图标离线打开。

## 本地运行

```powershell
python -m http.server 8000 --bind 0.0.0.0
```

同一局域网内的手机可通过电脑局域网 IP 和端口 `8000` 访问。

## 测试

```powershell
node --test
```

## 技术结构

应用为纯静态 HTML/CSS/JavaScript，不使用服务器、数据库、账号或云同步。`manifest.webmanifest` 提供手机安装信息，`sw.js` 负责离线缓存。
